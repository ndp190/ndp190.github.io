# Terminal Tips & Tricks

Here are some tips for using this terminal:

## Navigation

| Command | Description |
|---------|-------------|
| `help` | Show all available commands |
| `ls` | List files in current directory |
| `tree` | Show directory tree |
| `cat <file>` | Display file contents |

## Keyboard Shortcuts

- **Tab** or **Ctrl+I**: Autocomplete command
- **Up Arrow**: Previous command
- **Down Arrow**: Next command
- **Ctrl+L**: Clear terminal
- **Ctrl+U**: Clear current line

## Themes

Change the look with `themes set <name>`:

- dark
- light
- blue-matrix
- espresso
- green-goblin
- ubuntu
- nikk

Try `themes set blue-matrix` for a cool retro look!

## Pro Tips

> Use tab completion to save time typing commands.

Happy hacking!
