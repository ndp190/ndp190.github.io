## About This Site

This portfolio website is built as an interactive **terminal emulator** using:

- Next.js
- React
- TypeScript
- styled-components

## Why a Terminal?

I love the command line interface. It's fast, efficient, and feels like home for developers.

### Features

1. Command history with arrow keys
2. Tab autocomplete
3. Multiple color themes
4. File system navigation
