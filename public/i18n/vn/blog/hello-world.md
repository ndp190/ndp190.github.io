Cảm ơn bạn đã ghé thăm!

# Mẹo Sử Dụng website

Dưới đây là một số mẹo khi sử dụng website này:

## Điều Hướng

| Lệnh | Mô tả |
|------|-------|
| `help` | Hiển thị tất cả lệnh có sẵn |
| `ls` | Liệt kê file trong thư mục hiện tại |
| `tree` | Hiển thị cây thư mục |
| `cat <file>` | Hiển thị nội dung file |
| `themes` | Xem các theme có sẵn |
| `language` | Đổi ngôn ngữ (en/vn) |

> Sử dụng tab completion để tiết kiệm thời gian gõ lệnh.

## Phím Tắt

- **Tab** hoặc **Ctrl+I**: Tự động hoàn thành lệnh
- **Mũi tên lên**: Lệnh trước đó
- **Mũi tên xuống**: Lệnh tiếp theo
- **Ctrl+L**: Xóa terminal
- **Ctrl+U**: Xóa dòng hiện tại

## Chủ Đề

Thay đổi giao diện với `themes set <tên>`:

- dark
- light
- blue-matrix
- espresso
- green-goblin
- ubuntu
- nikk

## Ngôn Ngữ

Đổi ngôn ngữ với `language set <en|vn>`:

- `language set en` - Tiếng Anh
- `language set vn` - Tiếng Việt
